# Numerical Simulations in Python

This project implements numerical methods for solving ordinary differential equations (ODEs) and visualizing vector fields.

## Features
- Euler Method
- Runge-Kutta 4th Order (RK4)
- ODE Simulation
- Vector Field Visualization

## Technologies Used
- Python
- NumPy
- Matplotlib

## Example Problems

1. dy/dt = -2y
2. dx/dt = y
   dy/dt = -x

## How to Run

pip install -r requirements.txt
python main.py
